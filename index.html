Convert "NetLogo Hubnet" to "NetLogo Hubnet Web"

1. Find a NetLogo Hubnet file.
Run NetLogo 6.x.
Open "Gridlock Hubnet".
Save as "GridlockHubnet.nlogo"

2. Upload a NetLogo Hubnet file.

3. Receive Download of a NetLogo Hubnet Web zip file.

4. Run NetLogo Hubnet Web App locally.
Install node and npm.
Unzip the compressed NetLogo Hubnet Web file.
In a command line, navigate to the unzipped folder.
Type npm install to install dependencies.
This reads the package.json file and installs dependencies in a node-modules folder.
Type node server.js to run the server.
Open a browser and go to localhost:3000

4. Troubleshoot.
If you want to stop the server, type ctrl-c.
If you get an error when attempting to open the webpage hubnet-reset is not defined, then that command is not implemented in NetLogo Hubnet Web, and you should comment it out.
In index.html, find the NetLogo code and comment it out ;; hubnet-reset.
If you get this error: This non-standard character is not allowed., then look in a client CHOOSER and remove the escape character. Change \"b\" to "b"

5. Host NetLogo Hubnet Web App remotely.
Build and deploy on Heroku.com
